Projects

  • Playback
    • A music review website utilizes Spotify API to allow registered users to search, view, and review a selection of over a million albums. Backend utilizes Flask framework and MongoDB.
  • Terptales
    • A mobile e-reader app (iOS & Android compatible) which allows the user to flip through pages using their device’s gyroscope, annotate, and bookmark pages. Books can be added from local filesystem or the web.
  • Keysprint
    • A minimalistic typing test, emphasizing clean interface design. Users can assess and improve typing speed with distraction-free ease, leveraging intuitive tracking and performance monitoring features.
  • MicrOCaml
    • An interpreter for a dynamically-typed version of OCaml with a subset of its features. The lexer uses regular expressions to convert a string into a list of tokens. The parser checks if the list of tokens is grammatically correct and generates a tree. The evaluator processes the tree and produces output.
  • Ournix
    • A filesystem superficially similar to the Unix filesystem uses a dynamically-allocated linked data structure in C to manage files with functions touch, mkdir, cd, ls, pwd, rm, and rmfs.
